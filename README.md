## ⭐ Featured Project

### 📈 SR Hunter V4 — TradingView Pine Script Indicator

**SR Hunter V4** is my main featured project.

SR Hunter V4 is a layered Pine Script indicator built for TradingView.  
It helps validate trading setups through structured market analysis.

```text
Trend → Structure → Location → Trigger → Quality → Risk → Decision → UI
```

The goal of this project is not to generate random buy or sell signals.  
The goal is to help traders validate market conditions step by step before making a decision.

Repository:

🔗 [github.com/raafikaziiz-coder/tradingview-pine-scripts](https://github.com/raafikaziiz-coder/tradingview-pine-scripts)

---

## 🧠 SR Hunter V4 Concept

SR Hunter V4 is built around a layered validation workflow:

```text
Trend
↓
Structure
↓
Location
↓
Trigger
↓
Quality
↓
Risk
↓
Decision
↓
UI Output
```

### Core Philosophy

```text
No blind signal
No random entry
No emotional decision

Only structured validation
```
